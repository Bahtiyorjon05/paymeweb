## Deployment Steps

1. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "Prepare PaymeBot for Railway deployment"
   git push origin main
   ```

2. **Connect to Railway:**
   - Go to railway.app and sign in
   - Create new project
   - Select "Deploy from GitHub"
   - Choose your repository

3. **Set Environment Variables:**
   - Go to Variables tab in Railway dashboard
   - Add the required environment variables listed above

4. **Add Add-ons:**
   - Add PostgreSQL database
   - Add Redis for caching

5. **Deploy:**
   - Wait for build to complete
   - Your app will be accessible at the Railway URL

## Post-Deployment Setup

After your first deployment:

1. Access the Railway console
2. Run: `python manage.py migrate` (if not done automatically)
3. Create superuser if needed: `python manage.py createsuperuser`

## Admin Access
- Admin panel: `https://[your-project-name].up.railway.app/admin/`
- Use the credentials you set up
